📌 Project Overview

This project implements fundamental graph algorithms and demonstrates their real-world applications such as social networks, navigation systems, and network design.

---

🧠 Algorithms Implemented

- 🔹 Graph Representation (Adjacency List & Matrix)
- 🔹 Breadth First Search (BFS)
- 🔹 Depth First Search (DFS)
- 🔹 Topological Sorting
- 🔹 Dijkstra Algorithm (Shortest Path)
- 🔹 Bellman-Ford Algorithm (Negative Weights)
- 🔹 Minimum Spanning Tree (Prim’s Algorithm)

---

🌍 Real-World Applications

Algorithm| Application
BFS| Friend suggestion system
DFS| Network traversal
Topological Sort| Task scheduling (dependencies)
Dijkstra| Emergency route finding
Bellman-Ford| Routing with negative weights
MST (Prim)| Network cabling optimization
